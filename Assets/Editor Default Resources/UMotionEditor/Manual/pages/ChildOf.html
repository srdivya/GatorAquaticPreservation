Child-Of Professional

With the Child-Of Constraint it is possible to change the parent of a joint/transform during an animation. This is useful for pick-up, throwing or gun reloading animations.

The position and rotation of a joint/transform will be automatically updated with the movement of the new parent. Please note, that due to the fact that Unity uses a Vector3 for scaling, it is not possible for the Child-Of constraint to correctly distort a joint/transform when the parent is scaled and the child is rotated.

Only one Child-Of Constraint can be added per joint/transform.

Setup

Nothing needs to be setup.

Animated Properties

Property Description
Parent Selects a new parent. If none is selected, its original parent is used.
